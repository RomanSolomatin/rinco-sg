Rinco Static Generator

Rinco is a simple static generator that allows you to create a static site faster. Rinco has some syntax helpers to improve your development.


Github
Getting Started


Css Preprocessors

Pure CSS is great, but what if we can code in some preprocessor like sass or less? Better, what if we can code in all together? That is it, Rinco allows you to code in SASS, LESS and Stylus preprocessors in the same time, you are able to use a bunch of css library in differents preprocessors. Try it.


JavaScript Compilers




JavaScript is one of the most important languages nowadays, and there is many solutions that allows us to keep a pattern. Because of that, Rinco implements the principal javascript compilers, Coffeescript and Typescript. We know that ES6 is coming soon, so Rinco allows you to code in ES6 with Babel compiler as well. Amazing!

Template Languages

Sometimes we need to list informations from a json file on the screen, and we don't want to write each one because is too expensive, so we need a template language like Mustache to solve this problem. Now imagine that we want to write some articles, but we don't want to code in html because is too tiresome and difficult to read, in this case, we can use Markdown language. Great!
